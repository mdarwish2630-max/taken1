# TakPro Theme - دليل التثبيت
# ================================

## الخطوة 1: نسخ ملفات الثيم
انسخ مجلد `takpro` بالكامل إلى مسار `themes/` في المشروع:

```
cp -r takpro/ /path/to/your/project/themes/takpro/
```

## الخطوة 2: تنفيذ ملف SQL
استورد ملف SQL في قاعدة البيانات:

```bash
mysql -u username -p database_name < themes/takpro/takpro_theme.sql
```

أو يمكنك تنفيذ الأمر SQL يدوياً في phpMyAdmin أو أي أداة إدارة قواعد بيانات.

## الخطوة 3: تفعيل الثيم
بعد استيراد SQL، الثيم سيكون مفعّل تلقائياً (`is_active = 1`).

يمكن للمستأجرين اختيار الثيم من:
**لوحة التحكم → الإعدادات → تغيير الثيم**

## الخطوة 4: معاينة الثيم
افتح الرابط التالي في المتصفح:
```
https://your-domain.com/theme-preview/takpro
```

## ملفات الثيم

| الملف | الوصف |
|-------|-------|
| `_head.php` | رأس الصفحة (meta, خطوط, Tailwind, CSS) |
| `_navbar.php` | الشريط العلوي + قائمة التنقل |
| `_footer.php` | شريط التواصل + الفوتر |
| `_scripts.php` | زر واتساب + العودة للأعلى + JavaScript |
| `default.php` | الصفحة الرئيسية (Landing Page) |
| `about.php` | صفحة من نحن |
| `services.php` | صفحة الخدمات |
| `service.php` | صفحة تفاصيل خدمة واحدة |
| `contact.php` | صفحة تواصل معنا |
| `gallery.php` | صفحة معرض الأعمال |
| `takpro_theme.sql` | ملف SQL لتسجيل الثيم في قاعدة البيانات |

## المميزات
- ثنائي اللغة (عربي / إنجليزي)
- تصميم متجاوب (Responsive)
- ألوان برتقالية دافئة (#ff7a00)
- خط Cairo العربي
- دعم RTL/LTR
- تكامل كامل مع CMS (خدمات، بانرات، شهادات، إحصائيات، ميزات)
- زر واتساب عائم
- أنيميشن عند التمرير
- عداد أرقام متحرك
- دعم بيانات تجريبية عند عدم وجود بيانات
